# PDF Merger Script

This Node.js script merges two PDF files (`sample1.pdf` and `sample2.pdf`) into a single PDF called `Merged.pdf`. It also logs the number of pages in each input PDF and the total pages in the merged PDF.

---

## Prerequisites

- **Node.js** (v14 or higher recommended)  
- **npm** (comes with Node.js)

---

## Installation

1. Download or unzip the project folder.  
2. Open a terminal or command prompt in the project folder.  
3. Install the required dependency:

```bash
npm install pdf-lib



## Run the script using
node merge.js
